# [Pinely Round 1 (Div. 1 + Div. 2)](https://codeforces.com/contest/1761)

## A

两种可能

- $a = n, b = n$
- $n > a + b + 1$

## B

- 如果只有 2 个值，必然是交替的，所以最终答案就是 $\frac{n}{2} - 1$
- 否则，答案就是 $n$


## C

初始让 $s[i] = i$，然后拓扑排序即可

## D

看官方题解后发现一步一步分析其实也是能搞出来的，为什么不搞呢？

## E

首先用并查集找出连通块。

- 连通块为 1，答案为 0
- 否则，如果连通块有非完全图，那么找一个非割点，那么答案就是 1
- 否则，如果连通块为 2，那么就把小的全部搞到大的那边
- 否则，随便找两个连通块的点就行
